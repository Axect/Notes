1. Least Square Approach

1.1 Linear Modelling with two parameters

Let's start with two parameters.

  • Model :
(1)f(x;w0,w1)=w0+w1x

  • Squared loss function

    (2)Ln(tn,f(xn;w0,w1))=(tn−f(xn;w0,w1))2

  • Average loss

    (3)L=1N∑n=1NLn(tn,f(xn;w0,w1))

  • Best model: Choose w0,w1 which minimize average loss.

    (4)argminw0,w11N∑n=1NLn(tn,f(xn;w0,w1))

1.2. Least square solution

1.2.1 Theory

Let's expand average loss in terms of w1 :

(5)L=1N∑n=1NLn(tn,f(xn;w0,w1))=1N∑n=1N(tn−(w0+w1xn))2=1N∑n=1N(w12xn2+2w1xn(w0−tn)+w02−2w0tn+tn2)

By the same way, we can also describe L in terms of w0 :

(6)L=1N∑n=1N(w02+2(w1xn−tn)w0+w12xn2−2w1xntn+tn2)

Since L is polynomial of w0, w1, we can find extremum via differentiation.

(7)∂L∂w1=2w11N(∑n=1Nxn2)+2N(∑n=1Nxn(w0−tn))=0
(8)∂L∂w0=2w0+2w11N(∑n=1Nxn)−2N(∑n=1Ntn)=0

We want to find minimum - see 2nd order derivative.

(9)∂2L∂w12=2N(∑n=1Nxn2)≥0
(10)∂2L∂w02=2>0

Thus, we guarantee previous extrema are minima.

From (8), we can find optimal w0^:

(11)w0^=t―−w1x―

And also from (7), we can find optimal w1^:

(12)w1^=xt―−x―t―x2―−(x―)2

1.3. Linear Modelling with N parameters

1.3.1 Vector/Matrix notation

  • Linear model

    (13)f(xn;w0,w1)=w0+w1xn=wTxn

    where

    (14)w=[w0w1], xn=[1xn]

    Then we can extend to this model for N parameters.

    (15)f(xn;w)=w0+w1x1+⋯+wNxN=wTxn

  • Loss (Squared loss)

    (16)L=1N∑n=1N(tn−wTxn)2=1N(t−Xw)T(t−Xw)

    where

    (17)X=[x1T⋮xNT]=[1x1⋮⋮1xN], t=[t1⋮tN]

    And also, we can expand it.

    (18)L=1NwTXTXw−2NwTXTt+1NtTt

  • Least square solution with vector/matrix notation

    (19)w^=(XTX)−1XTt=X†t

1.4. Non-linear response from Linear model

1.4.1 Quadratic in a data

  • Quadratic model (Still linear for weight)

    (20)f(x;w)=w0+w1x+w2x2

  • Corresponding data matrix & target vector

    (21)X=[1x1x121x2x22⋮⋮⋮1xNxN2], t=[t1t2⋮tN]

  • Optimal weight

    (22)w^=[w0w1w2]=X†t

1.4.2 Polynomial in a data

  • Polynomial model (still linear for weight)

    (23)f(x;w)=∑k=0Kwkxk

  • Corresponding data matrix & target vector

    (24)X=[1x1⋯x1K1x2⋯x2K⋮⋮⋱⋮1xN⋯xNK], t=[t1t2⋮tN]

  • Optimal weight

    (25)w^=[w0⋮wK]=X†t

1.4.3 With basis function

  • Generic linear model with basis function (still linear for weight)

    (26)f(x;w)=∑k=0Kwkϕk(x)=wTϕ(x)

    where ϕk:RD→R, ϕ:RD→RK+1 such that ϕ0(x)=1.

  • Design matrix

    (27)Φ=[ϕ(x1)Tϕ(x2)T⋮ϕ(xN)T]=[1ϕ1(x1)⋯ϕK(x1)1ϕ1(x2)⋯ϕK(x2)⋮⋮⋱⋮1ϕ1(xN)⋯ϕK(xN)]

  • Optimal weight

    (28)w^=[w0⋮wK]=Φ†t

1.5 Generalization and Over-fitting

Let's see below figure.

  • Model complexity ↑ ⟹ Training loss ↓, Validation loss ↑
  • Thus, we should split our data to training set and validation set.
  • But if data is small, then how to solve this over-fitting problem?

 

1.5.1 Cross-validation

  • K-fold cross-validation splits the data into K equally sized blocks.
  • Each block takes its turn as a validation set for a training set comprised of the other K−1 blocks.

  • Averaging over the resulting K loss values gives us our final loss value.

An extreme case of K-fold cross-validation is where K=N. (N is the number of data)

This particular form of cross-validation is given the name Leave-One-Out Cross-Validation (LOOCV). The average squared validation loss for LOOCV is

(29)LCV=1N∑n=1N(tn−w^−nTxn)2

where w^−n is the estimate of the parameters without the n-th training example.

1.5.2 Regularization Least Square

Now, we choose another way that prevent over-fitting : Regularization.

Suppose polynomial model : f(x;w)=∑k=0Kwkxk

If all wk are zero, then there aren't any model complexity. And now, let w0 be any non-zero value and suppose w1≠0. Then we know the model complexity grows.

Thus, we could define the complexity of our model as

(30)∑iwi2=wTw

Then also modify our loss function.

(31)L′=L+λwTw

Then our optimal weight is as follows.

(32)w^=(XTX+NλI)−1XTt

Or with basis function,

(33)w^=(ΦTΦ+NλI)−1ΦTt
